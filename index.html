Generate ETL and SQL objects from object notation and C#

Daf can construct deployable ETL and SQL objects from simple object notation combined with C# code.

Run anywhere

Daf runs on Windows, Mac OS and Linux. All you need is Visual Studio Code and a set of plugins matching your target data architecture.

Code-from-scratch or deploy a complete Data Vault using Daf DW

You can code your own ETL flows or data warehouse from scratch using only Daf Core, or you can combine Core with Meta and DW (yet to be released) for a complete, fully automated Data Vault 2.0 solution.
